# Wanderbound

Polarsteps ZIP → print-ready photo album. FastAPI + Vue 3 monorepo.

## Operating Rules

- Use `mise run <task>` for repo tasks. Read @mise.toml for the task list.
- Prefer less code: builtins, canonical packages, and platform features first.
- Surface unrelated issues you notice. Fix them only when they block the task,
  are in touched code, or the user confirms broader cleanup.
- For dependencies, architecture, external APIs, standards, and tooling choices,
  verify current docs and practitioner usage before recommending or changing
  direction.
- For semantic code navigation, prefer `cymbal` over raw text search. Use normal
  shell tools for simple file listing, status checks, and exact-path reads.

## Non-Obvious Constraints

- All scripts live in `scripts/`. Run them through `mise run <task>`, not
  directly.
- Run `mise run generate` before `docker compose build` on a fresh clone.
- `frontend/src/client/` is generated by @hey-api/openapi-ts - never edit it.
- Album pages are A4 landscape (297×210mm) in CSS mm units.
- Preview and print must be visually identical. Never conditionally alter album
  visuals for print mode.
- Segments have composite PKs: `(uid, aid, start_time, end_time)`.
- PydanticJSON TypeDecorator round-trips JSON columns through Pydantic validation.
- Processing sessions replay all past SSE events on reconnect (no restart needed).
- Use pathlib, never os/os.path.
- Use rem, never px, except 1px for hairline borders, outlines, and optical
  nudges.
- Python 3.14: `except A, B:` is valid - no parentheses needed.
